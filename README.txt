A4 專業排版工具（手機優先 UI 版）

開啟方式：
1. 解壓縮
2. 直接打開 index.html

本版重點：
- 手機優先導覽列（設定 / 看板 / 預覽）
- 手機底部快捷操作列（加圖 / 文字卡 / 下載）
- 設定區改成可收合群組，手機更易操作
- 保留 IndexedDB 自動儲存與重設全部
- 保留拖曳排序、跨欄移動、圖片加字、A4 預覽下載

建議：
- 手機可直接使用 Safari / Chrome 開啟
- 若要長時間處理大量圖片，平板或桌面會更舒服


[v4]
- 看板預覽改為縮細高清預覽，使用 previewData 而非低像素 thumb。
- 拖曳卡片改成更接近參考圖的左縮圖 + 右資訊布局。
- 舊版自動存檔仍可載入；會自動回退使用舊 thumb / 原圖。


v10 修正：
- 手機全區域可左右滑動欄位（卡片區也可滑）
- 拖曳時減少陰影與 transform，降低卡頓
- 拖曳只在實際開始後才鎖 gesture
